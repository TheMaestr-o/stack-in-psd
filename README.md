# Stack in PSD

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Photoshop](https://img.shields.io/badge/Photoshop-CS6+-ff00ff)

Reverse layer stack and export to PSD format.

## Features

- Reverse layer order
- Remove file extensions from layer names
- Save as PSD to Desktop
- Automated workflow

## Installation

Copy to Photoshop Scripts folder

## Usage

File → Scripts → Browse → Select script
